字符串和数值转换

  • 字符串和数值转换
    • str - 数字=>字符串
    • int、float - 字符串=>数字
    • ord - 字符=>数字
    • hex - 数字=>十六进制字符串
    • float.fromhex - 十六进制字符串=>数字
    • oct - 数字=>八进制字符串
    • binascii - 二进制<=>十六进制字符串

str - 数字=>字符串

>>> str(100)
'100'

int、float - 字符串=>数字

>>> int('100')
100
>>> float('100.00')
100.0

ord - 字符=>数字

>>> ord('a')
97

hex - 数字=>十六进制字符串

>>> hex(97)
'0x61'

float.fromhex - 十六进制字符串=>数字

>>> float.fromhex('0x61')
97.0

oct - 数字=>八进制字符串

>>> oct(97)
'0141'

binascii - 二进制<=>十六进制字符串

>>> import binascii
>>> repr(binascii.a2b_hex('0099'))
"'\\x00\\x99'"
>>> binascii.b2a_hex('\x00\x99')
'0099'
