StatusBarView

该类是一个自定义的视图类，决定了状态栏的布局、属性等。

  • StatusBarView
    • 要点分析
    • 代码片断解析

要点分析

  • 状态栏实际上包含三个区域，NotificationIcon、StatusIcon和Date，分别标识通知区域，ICON区域和时间区域。
  • onLayout方法中处理子视图。
  • getViewOffset获取最左边的视图位置。
  • onTouchEvent和onInterceptTouchEvent处理触摸事件。

// 获取最左边视图位置
private int getViewOffset(View v) {
    int offset = 0;
    while (v != this) {
        offset += v.getLeft();
        ViewParent p = v.getParent();
        if (v instanceof View) {
            v = (View)p;
        } else {
            throw new RuntimeException(v + " is not a child of " + this);
        }
    }
    return offset;
}

// 处理触摸事件回调
public boolean onTouchEvent(MotionEvent event) {
    if (event.getAction() != MotionEvent.ACTION_DOWN) {
        mService.interceptTouchEvent(event);
    }
    return true;
}

// 阻止触摸事件回调
public boolean onInterceptTouchEvent(MotionEvent event) {
    return mService.interceptTouchEvent(event)
            ? true : super.onInterceptTouchEvent(event);
}
