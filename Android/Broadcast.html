相关类

import android.content.Intent;
import android.content.Context;
import android.content.BroadcastReceiver;

接受方

  • 继承BroadcastReceiver类，并实现onReceive方法

例程源码(JAVA) 

public class ConnectionStatus extends BroadcastReceiver {

    @Override
    public void onReceive(Context context, Intent intent) {
        // do something
    }
}

发送方

  • BroadcastReceiver对象作为Intent示例的一个参数传递，用来表示这个广播是一个行为
  • 通过Context类的sendBroadcast或者PendingIntent类的getBroadcast方法发送一个广播
  • 在manifest文件中还需要加入一个receiver配置项

例程源码(JAVA) 

Intent intent = new Intent(this, ConnectionStatus.class);
intent.setFlags(Intent.FLAG_ACTIVITY_NEW_TASK);
sendBroadcast(intent);

例程源码(XML) 

<receiver android:name=".ConnectionStatus" />

注册与注销

  • registerReceiver根据特定的过滤器注册一个BroadcastReceiver
  • unregisterReceiver注销一个BroadcastReceiver 
Intent registerReceiver(BroadcastReceiver receiver, IntentFilter filter, 
                        String broadcastPermission, Handler scheduler);
Intent registerReceiver(BroadcastReceiver receiver, IntentFilter filter);
