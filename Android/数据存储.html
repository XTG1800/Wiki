数据存储

  • Android支持四种数据存储方式，Preference、File、Database和ContentProvider
  • SharedPreference的数据存放路径为/data/data/PackageName/Shared_Pref
  • File的数据存放路径为/data/data/PackageName/files
  • Database的数据存放路径为/data/data/PackageName/database

SharedPreference

一种轻量级的数据存储方式，采用键值对的方式存储，主要用于应用程序配置，数据不能共享

  • 参见持久化

File

主要使用文件操作，用于比较大的文件，如图片

  • File存储
  • 外部存储的读写

Database

移动设备使用的小而强大的数据库，数据可以共享

  • SQLiteDatabase

ContentProvider

提供各种应用之间的数据共享机制，为数据访问提供一个统一的方法

  • ContentProvider
