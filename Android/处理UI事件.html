处理UI事件

相关方法与变量

abstract void onClick(View v)
处理点击动作回调方法
abstract boolean onLongClick(View v)
处理长按动作的回调方法
abstract void onFocusChange(View v, boolean hasFocus)
处理焦点变化的回调方法
abstract boolean onKey(View v, int keyCode, KeyEvent event)
处理按键的回调方法
abstract boolean onTouch (View v, MotionEvent event)
处理触摸状态回调方法
abstract void onCreateContextMenu(ContextMenu menu, View v, ContextMenu.ContextMenuInfo menuInfo)
处理Context菜单被弹出的回调方法

监听器接口

onClick()
接口对象 View.OnClickListener
onLongClick()
接口对象 View.OnLongClickListener
onFocusChange()
接口对象 View.OnFocusChangeListener
onKey()
接口对象 View.OnKeyListener
onTouch()
接口对象 View.OnTouchListener
onCreateContextMenu()
接口对象 View.OnCreateContextMenuListener

自定义View的事件处理

onFinishInflate()
当View中所有的子控件均被映射成xml后触发
onMeasure(int, int)
确定所有子元素的大小
onLayout(boolean, int, int, int, int)
当View分配所有的子元素的大小和位置时触发
onSizeChanged(int, int, int, int)
当view的大小发生变化时触发
onDraw(Canvas)
view渲染内容的细节
onKeyDown(int, KeyEvent)
有按键按下后触发
onKeyUp(int, KeyEvent)
有按键按下后弹起时触发
onTrackballEvent(MotionEvent)
轨迹球事件
onTouchEvent(MotionEvent)
触屏事件
onFocusChanged(boolean, int, Rect)
当View获取或失去焦点时触发
onWindowFocusChanged(boolean)
当窗口包含的view获取或失去焦点时触发
onAttachedToWindow()
当view被附着到一个窗口时触发
onDetachedFromWindow()
当view离开附着的窗口时触发，Android123提示该方法和 onAttachedToWindow() 是相反的。
onWindowVisibilityChanged(int)
当窗口中包含的可见的view发生变化时触发
