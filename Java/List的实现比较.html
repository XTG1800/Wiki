List的实现比较

实现类比较表

实现类 实现方式 内存占用 同步问题 使用策略 特殊功能
ArrayList 数组 如果集合中的元素数目大于目前的数组元素长度，会动态增长50%，并把旧数据复制到新数组中，造成大量对象等待内存回收 不同步 适用于频繁的索引操作，但不适用于频繁的插入和删除操作  
LinkedList 链表 基于链表实现，因此添加和删除时不需要扩展大的内存和数据的复制操作 不同步 适用于频繁的插入和删除操作，但不适于频繁的索引操作  
Vector 数组 如果集合中的元素数目大于目前的数组元素长度，会动态增长100%，并把旧数据复制到新数组中，造成大量对象等待内存回收 同步 线程安全的类，但是操作速度较慢  
Stack 数组 如果集合中的元素数目大于目前的数组元素长度，会动态增长100%，并把旧数据复制到新数组中，造成大量对象等待内存回收 同步 实现后进先出的栈
